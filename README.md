# sistema-estudiantes
Sistema de Gestión de Estudiantes

Este proyecto es un programa en Python desarrollado en Google Colab, que permite administrar una lista de estudiantes usando un menú en la consola. Su función es mostrar, agregar, eliminar y actualizar estudiantes de manera sencilla. Fue creado como práctica para manejar listas, diccionarios y funciones en Python.

Funcionalidades del sistema
1. Mostrar estudiantes
Muestra a todos los estudiantes registrados con su:
-ID.
-Nombre.
-Grado.
-Materias favoritas.

2. Agregar estudiante
Permite añadir estudiantes nuevos proporcionando su nombre, grado y materias favoritas. El ID se genera automáticamente.

3. Eliminar estudiante
Elimina a un estudiante usando su ID. Si el ID no existe, el sistema lo indica.

4. Actualizar estudiante
Permite modificar nombre, grado y materias. Si se deja un campo vacío, se mantiene el dato anterior.

5. Salir
-Cierra el sistema.
-Tecnologías usadas.
-Python 3.
-Listas.
-Diccionarios.
-Funciones
-Entrada por consola

Objetivo del proyecto
El objetivo es practicar programación básica y aprender a manejar información usando estructuras de datos en Python. También sirve para entender cómo funcionan los menús interactivos dentro de un programa.
